# Smart Internship Tracker
A simple frontend web application to track internship and placement applications.


## Features
- Add internship or job applications
- Track company, role, stage, result, and applied date
- View all applications in a table
- Automatically generate summary statistics
- Delete applications using unique IDs
- Data persistence using browser localStorage
  

## Tech Stack
- HTML
- CSS
- JavaScript

## Live Demo
https://smart-internship-tracker.vercel.app/

## How to Run Locally
1. Clone the repository
2. Open `index.html` in a browser

## Use Case
This project helps students manage and track their internship and placement applications in a structured way.
